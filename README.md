# Tamizhan Abroad — Official Site 🤠🎮

Personal site for the Tamizhan Abroad YouTube channels and Instagram.

- **Vlogs:** https://www.youtube.com/@TamizhanAbroad
- **Gaming:** https://www.youtube.com/@TamizhanAbroadGaming
- **Instagram:** https://www.instagram.com/tamizhan__abroad/

## Structure
- `index.html` — the entire site (HTML + CSS + JS)
- `assets/` — logos, photo, favicon

## Editing
- Featured video / channel IDs: edit the `CONFIG` object near the bottom of `index.html`
- Instagram feed: search for `instagramFeed` in `index.html` and paste a widget embed code (behold.so / snapwidget.com)

Hosted on GitHub Pages.
